# Sistema de Notas 📝

Um sistema prático e eficiente desenvolvido em **C++** para gerenciamento de notas escolares, cálculo de médias, classificação de alunos e persistência de dados em arquivos.

---

## 🚀 Funcionalidades

O projeto foi construído focando na organização e armazenamento de dados, contando com as seguintes features:

* **Cálculo de Notas e Médias:** Sistema automatizado para processar as notas e calcular as médias dos alunos.
* **Classificação Automatizada:** Definição do status do aluno com base nas médias calculadas.
* **Persistência de Dados (Salvamento):** Gravação automática das informações em arquivos para não perder nenhum dado ao fechar o programa.
* **Leitura de Arquivos:** Carregamento do histórico de notas diretamente dos arquivos salvos ao iniciar o sistema.
* **Geração de Relatórios:** Exportação dos dados consolidados em um arquivo de texto limpo e legível (`relatorio.txt`).

---

## 📁 Estrutura do Projeto

* `main.cpp`: Código-fonte principal com toda a lógica do sistema.
* `relatorio.txt`: Relatório gerado com a classificação e notas dos alunos.
* `.gitignore`: Configuração para impedir que arquivos temporários ou executáveis (como o `main.exe`) sejam enviados ao GitHub!
